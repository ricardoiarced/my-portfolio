Case Study · 01

Personal Finance Tracker

A private workspace for recording financial activity, planning ahead, and understanding cash flow and net worth in one place.

Next.js · TypeScript · Tailwind · Supabase · PostgreSQL

The problem

Income, expenses, budgets, savings, investments, and liabilities affect one another, but separate tools make the current financial position difficult to understand.

The solution

I designed and built an authenticated application that keeps exact financial calculations, reporting boundaries, and user-owned data authorization explicit across the interface and persistence model.
